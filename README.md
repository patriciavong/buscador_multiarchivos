📂 Buscador Local de DocumentosEste proyecto es una herramienta web de procesamiento 100% local diseñada para buscar términos específicos dentro de múltiples archivos simultáneamente sin comprometer la privacidad de los datos.  🚀 CaracterísticasPrivacidad Total: Los archivos no se suben a ningún servidor; todo el procesamiento ocurre en el navegador del usuario.  Multiformato: Soporta lectura de archivos PDF, Excel (XLSX), Word (DOCX) y Texto Plano (TXT).  Conteo de Coincidencias: Identifica cuántas veces aparece el término buscado en cada documento cargado.  Interfaz Intuitiva: Soporte para arrastrar y soltar (Drag & Drop).  🛠️ Tecnologías utilizadasHTML5 / CSS3 / JavaScript (Vanilla).  PDF.js: Para la extracción de texto en documentos PDF.  SheetJS (XLSX): Para el procesamiento robusto de hojas de cálculo.  Mammoth.js: Para la conversión de archivos .docx a texto plano.  📋 Requisitos e InstalaciónNo requiere base de datos ni servidores complejos.Clona este repositorio o descarga los archivos.Asegúrate de que las librerías necesarias estén en la carpeta /librerias.  Abre index.html en tu navegador favorito.
